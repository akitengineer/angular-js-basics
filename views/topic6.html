Advantages:
  • MVC ( Model View Controller)
  • Data Models are POJO (Plain Old Javascript Object)
  • Behavior with directives.
  • Filters
  • Services
  • Less code or boiler plates.
  • Easy dom manipulation
  • Easy to Test
Disadvantages:
  • It does not fit with those libraries which are not angular specific.
Limitations:
  • Specifically designed for CRUD applications.
  • Not preferable for games development.
